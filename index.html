National Parks Passport

Technologies: Python - Django Rest Framework, JSON, HTML/CSS, Bootstrap, SQLite, jQuery, BCrypt

Ever wanted to know which National Parks you've visited, and which to add to your bucket list? This is the app for you. With the National Parks Passport, users can register and view relevant data about each of the 60 major US National Parks. See a park that you've visited? Add that park to your passport, and check your total against the leaderboard. So far the user with the most parks visited has 42!

I especially enjoyed building this app because beforehand I always had to count on my fingers how many parks I have visited. The framework used was Python Django, and the data for the parks and users is stored using SQLite.

JQZ Events

Technologies: C# - Asp.NET Core, HTML/CSS, Bootstrap, MySQL, jQuery, Microsoft Identity, Github, System.Net.Mail

Time to get that calendar in order! With the JQZ Events app, users have the ability to check their calendar, add events, and set reminders. Not only that, but they can also lookup friends using the app by email, send event invitations, RSVP to their others' friend request and invitations, and schedule free time.

I am very happy with the final result of this C# Asp.NET app. The major challenges came from building out a user friendly calendar, ensuring events wouldn't overlap, and coding a function to send out on time reminders via email. As it was a collaborative project, we relied heavily on Github for version control and code sharing as well as clear communication and teamwork to get the final product.

Collaborators: Quinn Lathe, Zack Springer, John Albert

JQZ Developer Playground

Technologies: MERN Full-stack - MongoDB, Express, React, Node.js, HTML/CSS, Bootstrap, MaterialUI

Show your stuff off on the Developer Playground app. This site lets users build a custom portfolio, share projects, view other projects and portfolios, solve algorithm challenges from a database of puzzles, as well as have some fun with a few games.

This app, built in the MERN full-stack framework, really put our JavaScript skills to the test. Being that it was a single page application (SPA), we had to pay special attention to each component and it's place in the file structure. We also employed Express.js to make api calls to a NoSQL Database, MongoDB. All in all, a really fun application to build!
